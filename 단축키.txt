컨트롤 + a => 코드 전체선택

컨트롤 + 시프트 + G => #include 파일로 이동

컨트롤 + k + o => Header <=> Cpp 전환 

컨트롤 + k + f => 코드 탭 and 줄정리

컨트롤 + k + c => 범위 주석

컨트롤 + k + u => 주석 제거

컨트롤 + delete => 한줄 제거

컨트롤 + 방향키 => 의미있는 곳으로 이동.

컨트롤 + 스페이스 => 자동완성 기능 켜기.

end => 줄 끝으로 이동

home => 줄 처음으로 이동

시프트 + home => 맨뒤에서 맨앞까지 선택

시프트 + end => 맨앞에서 맨뒤까지 선택

컨트롤 + 시프트 + 방향키  => 의미있는 프로그래밍 키워드별 선택

알트 + 시프트 + 방향키  => 다중 라인 선택

컨트롤 + 시프트 + U => 소문자 => 대문자

컨트롤 + U => 대문자 => 소문자

컨트롤 + r + r => 연관된 명칭만 변경
